# P2P Чат - Лабораторная работа 3

Одноранговый чат для обмена текстовыми сообщениями в локальной сети.

## Запуск программы

Программа запускается из командной строки с указанием имени пользователя и IP-адреса:

```bash
python3 p2p_chat.py --ip <IP_АДРЕС> --name <ИМЯ> [--port <ПОРТ>]
```

**Аргументы:**
- `--ip` — локальный IP-адрес для привязки (обязательно).
- `--name` — имя пользователя в чате (обязательно).
- `--port` — TCP-порт для прослушивания (по умолчанию 5000, опционально).

### Пример запуска нескольких узлов на одном компьютере

Для тестирования на одном компьютере необходимо использовать разные адреса loopback-интерфейса.

**1. Настройка дополнительных адресов (только для Linux):**
```bash
sudo ip addr add 127.0.0.2/8 dev lo
sudo ip addr add 127.0.0.3/8 dev lo
```
*(На Windows дополнительные адреса 127.0.0.x доступны по умолчанию)*

**2. Запуск трех экземпляров программы в разных терминалах:**

```bash
# Терминал 1
python3 p2p_chat.py --ip 127.0.0.1 --name User1

# Терминал 2
python3 p2p_chat.py --ip 127.0.0.2 --name User2

# Терминал 3
python3 p2p_chat.py --ip 127.0.0.3 --name User3
```

После запуска узлы автоматически обнаружат друг друга через UDP-broadcast и установят TCP-соединения.

## Использование

- **Отправка сообщения:** введите текст и нажмите `Enter`.
- **Просмотр подключенных узлов:** введите `/peers`.
- **Выход из чата:** введите `/quit`.

Узлы автоматически обмениваются историей сообщений при подключении нового участника.
